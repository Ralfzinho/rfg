# Visão Geral

O projeto **Race for Glory** é um sistema web para gerenciamento de um campeonato de automobilismo, inspirado na Fórmula 1. Ele permite a administração de equipes, pilotos, corridas e a publicação dos resultados e classificações.

## Funcionalidades Principais

*   **Páginas Públicas:**
    *   Página inicial com destaques e informações sobre o campeonato.
    *   Classificação de pilotos e equipes.
    *   Lista de pilotos e equipes da temporada.
    *   Calendário de corridas com detalhes de cada evento.

*   **Painel de Administração:**
    *   Sistema de login seguro para administradores e editores.
    *   Gerenciamento (CRUD) de:
        *   Pilotos
        *   Equipes
        *   Corridas
        *   Resultados
    *   Visualização da classificação geral.

## Tecnologias Utilizadas

*   **Backend:** PHP
*   **Frontend:** HTML, CSS (Tailwind CSS), JavaScript
*   **Banco de Dados:** MySQL
